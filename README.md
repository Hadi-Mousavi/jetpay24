# Jet Pardakht

**Jet Pardakht** is a modern financial services platform designed to simplify international payments and financial processes for Iranian students, professionals, and businesses.

The platform provides a secure workflow for service requests, identity verification, payment processing, document management, customer communication, and order tracking through a modern web application.

> **Official Website:** https://jetpardakht.com

---

## 🔄 Order Workflow

```text
Order Created
      │
      ▼
Waiting for Payment
      │
      ▼
Payment Review
      │
      ▼
In Review
      │
      ▼
In Progress
      │
      ▼
Waiting for Customer
      │
      ▼
Completed
```

Workflow recovery is supported for administrative mistakes while preserving a complete audit history.

## 🔐 Security

Jet Pardakht follows a security-first architecture.

Implemented protections include:

* Secure authentication
* Authorization & ownership validation
* Private file storage
* Magic-byte file validation
* Upload allow-list
* Upload block-list
* File size validation
* Rate limiting
* Audit logging
* Timeline history
* Secure download endpoints
* Notification system
